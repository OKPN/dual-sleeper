# Dual Sleeper v1.0.0 (デュアル・スリーパー)

**Dual Sleeper v1.0.0** は、Windows PC（特にAI学習サーバーやリモートPC）に最適な、知的で段階的な省電力（電源・モニター管理）システムです。

単なる「時間経過によるスリープ」ではなく、**「ユーザーの物理的な操作」**、**「ネットワークの通信速度（中央値集計）」**、**「WASAPI オーディオセッション（通話中検知）」**および**「GPU（ビデオカード）の動作プロセス」**の多元的な軸をインテリジェントに監視し、状況に応じた最適な省電力を自動で実行します。

---

## 💡 具体的なユースケースと動作

本アプリは、日常や開発における以下のような複雑な状況をインテリジェントにハンドリングし、的確にモニターを消灯してPCをスリープさせます。

1. **ただ放置しているとき（通常の離席）**
   * 👉 操作がなく、通信も発生しないため、無操作（デフォルト5分）で画面を消灯し、その5分後にPC本体をスタンバイ（スリープ）にします。
2. **LINE / Discord / Zoom 等で通話しているとき（寝落ち通話・画面最小化通話）**
   * 👉 **【WASAPI オーディオセッション保護】** 画面を最小化していたり、寝落ちして自分が無言になろうが、通話中である限りスリープに入りません（画面はすっきり消灯して省エネ維持）。通話が切断された瞬間にタイマーが再開し、安全に自動スリープします。
3. **YouTubeなどの動画サイトを見ているとき（視聴中）**
   * 👉 物理操作はなくても、動画再生によるバッファ通信や点灯延長タイトルを検知し、モニター消灯およびPC本体のスリープを防ぎます。
4. **大容量ファイルをダウンロードしているとき（放置ダウンロード）**
   * 👉 操作がないためモニターは電気代節約のために先行して消灯しつつ、Downloadsフォルダ内の一時ファイル（`.crdownload`, `.part`）検知により、ダウンロードが完了するまでPC本体はスリープせずに稼働し続けます。
5. **ゲームを起動したまま放置しているとき（寝落ち・キャラ放置）**
   * 👉 操作がないためモニターは消灯し、GPU使用率が 30% 以上であっても保護対象（python等）ではないため、**`🎮 ゲーム放置中`** と判定して安全にスリープを実行し、無駄な電気代をカットします。（※FF14等の放置バザー等は、保護プロセスに追加することで画面オフのまま放置継続も可能です）
6. **ゲームをしていて、かつ配信など重いネットワーク送信が重なっているとき（ゲーム配信中）**
   * 👉 モニターが消灯しても、配信データ（5 Mbps以上の高トラフィック）を検知し、**ゲーム配信中と判断してPC本体のスリープを確実に阻止**します。
7. **NVIDIA GPUでAI学習（LoRA学習やローカルLLM推論）などをしているとき**
   * 👉 操作がなく通信が低くても、GPUを消費しているプロセス名に `python.exe`（または `llama-server` 等）が含まれるため、**AI学習/推論中と判定してスリープを自動で阻止**します。

## 🌟 主な機能

### 📦 0. 【完全ポータブル化】Python未インストールのPCでも1秒起動
*   **ポータブルPython（Embedded Python 3.11）同梱:**
    本アプリには軽量なWindows用組み込みPython環境が全パッケージ同梱されています。
*   相手のPCに Python がインストールされているか、PATHが通っているかを気にする必要が**完全にゼロ**です。`run.bat` をダブルクリックするだけで、依存ライブラリの構築待ち時間もなく100%一瞬で安全に起動します。

### 🎙️ 1. 【WASAPI オーディオ保護】通話中・音声ストリーム中の完全保護
*   **Windows Core Audio API 直叩き:**
    Windowsの内部オーディオセッションを常時監視します。Discord、LINE、Zoom、ブラウザ等でマイクやスピーカーの音声ストリームがアクティブな場合、**消灯中（State 2）であってもスリープタイマーを自動でリセット（スリープ保護）** します。
*   **寝落ち通話への強さ:**
    通話中に自分が寝落ちして完全無音になっても、通話セッションが生きている限りPCはスリープしません。通話が切断された瞬間にタイマーが回り始め、5分後にPCが自動スリープに入ります。

### 📱 2. 通知 ＆ リモート操作（Discord Webhook ＆ Telegram Bot）
用途に合わせてお好みの通知・操作手段を選択できます。
*   **Discord Webhook (おまけお手軽機能 / Webhook URLを貼るだけ):**
    Webhook URLを設定するだけで導入できる手軽な通知機能です。スリープ予告通知の文面に **etherwake 専用 WoL 遠隔起動リンク (`wol_url`)** を含めることができるため、普段使いの Discord アプリで通知を受け取り、リンクをタップするだけで自宅PCをワンタップ遠隔起動できます。
*   **Telegram Bot (さらに高機能な双方向コントロール):**
    スマホからさらに高度なリモート操作を行いたい方向けのフル機能連携です。スリープ予告30秒前通知に対してスマホから1文字返信するだけで「画面消灯のまま10分スリープ延長」ができるほか、`/sleep`（電源予約トグル）、`/server`（サーバモード切替）、`/status`（詳細稼働確認）をスマホからインタラクティブに操作できます。

### 🔗 3. 【etherwake 専用 WoL連携】スリープ通知からスマホ1タップで遠隔起動
*   **etherwake 専用 Wake on LAN (WoL) 遠隔起動リンクの自動埋め込み:**
    `config.json` の `"wol_url"` に、制作者が運営するWeb WoL送信サービス **[etherwake.k7m.f5.si](https://etherwake.k7m.f5.si)** で発行した自身の起動用URLを設定しておくと、Discord や Telegram に届くスリープ予告通知の最下部に「WoL遠隔起動リンク」が自動挿入されます。

### 📊 4. 【通信統計】スパイクに騙されない「中央値 (Median)」計測
*   **外れ値（高速バースト通信）の無効化:**
    写真やWebページの瞬間的な爆発ロード（10 MB/sなど）が発生しても、算術平均のように数値が引っ張られることがありません。
*   集計データの中央値（Median）をベースに平常時の通信速度を算出するため、自身の環境における**「素のバックグラウンド通信ノイズ」**が100%正確に可視化されます。

### 🤖 5. AI学習とゲーム放置の自動判別（GPU保護機能）
*   **LoRA学習中（スリープ阻止）:**
    NVIDIA製GPUの負荷を監視します。GPUを使用しているプロセス名に `python.exe` や `llama-server` が含まれる場合は、AI利用中とみなしてスリープを自動阻止します。
*   **ゲーム放置中（スリープ実行）:**
    GPU使用率が 30% 以上であっても、保護対象外の場合は **`🎮 ゲーム放置中`** と識別して安全にスリープを実行します。

### ⚡ 6. 【落雷防災アラート】自宅周辺の雷を検知して即座に休止保護
*   **Open-Meteo API による5分周期監視 (APIキー不要/初期無効):**
    設定ファイルに自宅の位置情報を登録して `enabled: true` にすると、5分ごとに周辺の雷雨・落雷データを自動チェックします。
*   **📍 距離(km) ＆ 16方位（北西・南南西など）の自動解析通知:**
    自宅と観測ポイントの緯度・経度から **「自宅から見た方角（16方位）」** と **「直線距離（km）」** を数学的（ハバーサイン公式）にリアルタイム解析！
    `📍 検知位置: 自宅から【北西 約 14.2 km】地点` のように超具体的に通知されるため、風上からの雷雲接近かどうかがスマホで一目瞭然です。
*   **雷接近時の Telegram 通知 ＆ 休止トグル / 3ステップ自動休止モード:**
    周辺で雷雨・落雷が発生した際、Telegram へ `⚡️ 【落雷警報アラート】` を即座に通知します。
    `"auto_hibernate"` オプションにより以下の3つの動作モードから自由に選択可能です：
    *   **`"off"`** *(初期値)*: 常にスマホ通知を送り、ユーザーの返信（`1` や `h`）で休止状態へ移行。
    *   **`"state2_only"`**: 通常作業中（State 0/1）はスマホ通知で手動確認し、**画面消灯・放置中（State 2）の時だけ問答無用で即座に「休止状態」へ自動移行！**
    *   **`"always"`**: 作業中・放置中を問わず、雷検知で常に問答無用で即座に「休止状態」へ自動移行。
*   **位置情報の簡単な指定方法 (Googleマップ コピペ対応):**
    Googleマップで自宅の位置を右クリックしてコピーした文字列（例: `"35.6812, 139.7671"`）を `config.json` の `location` 項目にそのまま貼り付けるだけで自動パースされます。

## 💡 Wake on LAN (WoL) との最強の組み合わせ推奨

**Dual Sleeper** は、**WoL (Wake on LAN)** 技術と組み合わせることで、電気代を極限まで節約した完璧な「プライベート・リモート開発サーバー / リモートPC」を構築することができます。

### 🌐 制作者提供の Web WoL サービス: etherwake.k7m.f5.si
本アプリのWoL連携機能は、制作者が運営・提供するWeb WoL送信サービス **[etherwake.k7m.f5.si](https://etherwake.k7m.f5.si)** の専用機能です。

1.  **`config.json` への設定方法:**
    `https://etherwake.k7m.f5.si` で生成した自身の起動用URLを `config.json` の `"wol_url"` 欄に登録します。
    ```json
    "wol_url": "https://etherwake.k7m.f5.si/?data=..."
    ```
2.  **スマホ通知からのワンタップ起動:**
    PCがスリープに入る直前、Discord や Telegram に届く通知の最下部に上記のWoLリンクが自動的に掲載されます。
    PCを起こしたくなった時は、通知内のリンクをタップするだけでどこからでも一瞬で自室のPCを起動できます。

## 📂 ファイル構成

*   `dual_sleeper.py`: メインのPythonプログラム。
*   `python_embed/`: **[ポータブル環境]** 同梱された軽量Python 3.11環境。
*   `run.bat`: 同梱Pythonを呼び出してダブルクリック一発で起動するバッチファイル。
*   `config.json.example`: 設定ファイルのひな形。
*   `config.json`: **[ローカル専用]** 実際の動作設定ファイル（`.gitignore` により自動除外）。
*   `requirements.txt`: 動作に必要な外部ライブラリ（`psutil`）。
*   `test_webhook.bat`: **[テスト用]** Discord / Telegram通知の導通テスト用バッチファイル。

## ⚙️ 設定オプション (`config.json`)

| 設定項目 | 初期値 | 単位 | 説明 |
| :--- | :--- | :--- | :--- |
| `idle_limit_seconds` | `300` | 秒 | モニター消灯を検討し始める無操作時間（5分）。 |
| `network_limit_kbs` | `20.0` | KB/s | 通常の無通信状態とみなす通信速度のしきい値。 |
| `network_check_duration_seconds` | `30` | 秒 | 低通信状態が何秒続いたらモニターを消灯するか（30秒）。 |
| `check_interval_seconds` | `5` | 秒 | 稼働状態のチェック周期。 |
| `standby_after_monitor_off_seconds` | `300` | 秒 | モニター消灯後、PC本体をスリープさせるまでの待機時間（5分）。 |
| `game_gpu_threshold_percent` | `30` | ％ | ゲーム等のGPU使用放置とみなすGPU使用率の閾値。 |
| `high_network_limit_kbs` | `625.0` | KB/s | ゲーム配信中などの高トラフィックとみなすしきい値（5 Mbps相当）。 |
| `keep_awake_window_titles` | `["youtube:20", "twitch", "zoom:60", "obs:360"]` | リスト | 点灯延長を行うウィンドウタイトルと時間（分）。 |
| `server_mode` | `"off"` | 文字列 | 高速消灯サーバモード（`"off"`, `"desktop"`, `"always"`）。 |
| `discord_webhook_url` | `""` | URL | Discord Webhook URL（お手軽通知機能）。 |
| `telegram_bot_token` | `""` | トークン | Telegram Bot API Token（高機能双方向操作用）。 |
| `telegram_chat_id` | `""` | ID | Telegram 宛先チャットID。 |
| `wol_url` | `""` | URL | スリープ通知内に自動挿入する etherwake 専用の Wake on LAN 遠隔起動リンク（制作者運営の **`https://etherwake.k7m.f5.si`** のURL）。 |
| `sleep_pending_seconds` | `30` | 秒 | スリープ予告通知後の猶予時間。 |
| `lightning_protection` | `{}` | オブジェクト | 落雷防災アラート設定。`enabled`（有効化）、`location`（Googleマップ位置コピペ）、`auto_hibernate`（`"off"`, `"state2_only"`, `"always"` モード）。 |
